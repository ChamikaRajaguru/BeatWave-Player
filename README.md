# 🎵 BeatWave – MP3 Player with Equalizer

BeatWave is a Flutter-based music player application that allows users to play MP3 files with built-in equalizer controls for a customizable audio experience. The app focuses on smooth performance, clean UI, and enhanced sound control.

---

## 🚀 Features

- 🎧 Play local MP3 audio files
- 🎚️ Built-in Equalizer for sound customization
- ⏯️ Play, Pause, Next, Previous controls
- 📂 Browse and manage music files
- 🎨 Clean and user-friendly UI
- ⚡ Smooth and responsive performance

---

## 🛠️ Built With

- Flutter (Dart)
- Audio playback libraries (e.g., just_audio / audioplayers)
- Equalizer / DSP integration
